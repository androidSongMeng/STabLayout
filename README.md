## How to use?
 #### Step 1. Add the JitPack repository to your build file
```java
  allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
#### Step 2. Add the dependency
```java
  dependencies {
	        compile 'com.github.androidSongMeng:StreamList:v1.0.1'
	}
```
